# 🏫 SMK Negeri 3 Balige — Frontend Web

## Cara Menjalankan

Pastikan NestJS backend sudah berjalan dulu di `http://localhost:3000`, lalu:

```bash
npm install
npm run dev
```

---

## 🌐 Halaman Public

| Path | Halaman |
|------|---------|
| `/` | Beranda |
| `/berita` | Berita & Informasi |
| `/profil` | Profil Sekolah |
| `/portofolio` | Portofolio & Skill |

---

## 🔐 Halaman Admin Panel

| Path | Halaman |
|------|---------|
| `/admin` | Dashboard |
| `/admin/sejarah` | Sejarah & Identitas |
| `/admin/visi-misi` | Visi & Misi |
| `/admin/struktur` | Struktur Organisasi |
| `/admin/program` | Program Keahlian |
| `/admin/fasilitas` | Fasilitas & Sarana Belajar |
| `/admin/prestasi` | Akreditasi & Prestasi |
| `/admin/mitra` | Mitra Kerjasama |

---

## 🔗 URL Lengkap (Development)
